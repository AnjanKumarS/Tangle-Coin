TangleCoinD - Blockchain Simulation

TangleCoinD is a secure and scalable cryptocurrency platform using a combination of **Directed Acyclic Graph (DAG)** and **Merkle Trees** for efficient transaction processing.

Key Features:

  • Merkle Trees: Ensures secure transaction verification using SHA-256 hashing.
  • DAG: Enables parallel transaction processing, increasing scalability and reducing congestion.
  • Proof of Stake (PoS): Used for validator selection to maintain network integrity and reduce energy consumption.
  • Fast Transaction Speed: DAG architecture supports simultaneous transaction confirmations, improving performance.
